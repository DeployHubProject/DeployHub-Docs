  Generate a random number. 
---

Returns a random number between 0 and the specified limit.

**Usage:**

random(limit)

**Returns:**

Integer: A number between 0 and limit-1.

**Example**

echo "Here's a random number between 0 &amp; 50 $(random(51))";
