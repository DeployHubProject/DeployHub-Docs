### me

me returns a _User Object_ representing the current logged in user.

**Usage:**

me()

**Returns:**

User Object

**Examples:**

set me = me();

echo "Hello ${me.name}";


Further descriptions of User Objects are found in the high level section named _DMScript_ Objects, in the section named User.
