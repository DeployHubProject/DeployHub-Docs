  Convert to uppercase.
---

_upper_ returns an uppercase version of the passed string.

**Usage:**

String.upper()

Or

upper(String)

**Returns:**

String

**Examples:**

```bash
set a = "Hello There";

echo ${a.upper()};

Result:

HELLO THERE

**See Also:**

lower()
```
