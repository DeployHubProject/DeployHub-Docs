### upper

upper returns an uppercase version of the passed string.

**Usage:**

String.upper()

Or

upper(String)

**Returns:**

String

**Examples:**

set a = &quot;Hello There&quot;;

echo ${a.upper()};

Result:

HELLO THERE

**See Also:**

lower()
